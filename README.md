# A* Pathfinding Algorithm Visualization

## Описание проекта

Данный проект представляет собой интерактивную визуализацию алгоритма поиска пути A* (A-Star) на двумерной сетке с препятствиями. Программа позволяет пошагово наблюдать за работой алгоритма, видеть, как выбираются узлы, и анализировать значения G, H и F-стоимостей для каждой клетки.

## Особенности

- **Пошаговая визуализация** – каждый шаг алгоритма выполняется по нажатию клавиши Enter
- **Несколько режимов отображения**:
  - **Simple** – визуализация с символами (S - старт, G - цель, @ - текущий узел, ? - открытые узлы, . - посещенные узлы, * - найденный путь)
  - **G-cost** – отображение стоимости пути от старта до каждой клетки
  - **H-cost** – отображение эвристической оценки расстояния до цели
  - **F-cost** – отображение полной оценочной стоимости (G + H)
- **Детальная информация**:
  - Список соседей текущего узла с их стоимостями
  - Отсортированный список открытых узлов (open list) с выделением лучшего кандидата
  - Статистика: номер шага, количество открытых и закрытых узлов
- **Визуализация пути** – после завершения алгоритма найденный путь подсвечивается зеленым цветом во всех режимах
- **Лабиринт** – содержит туннели и открытые комнаты для наглядной демонстрации работы алгоритма

## Управление

| Клавиша | Действие |
|---------|----------|
| Enter | Следующий шаг алгоритма |
| G | Переключить на режим G-cost |
| H | Переключить на режим H-cost |
| F | Переключить на режим F-cost |
| S | Переключить на простой режим (Simple) |
| R | Перезапустить с новыми координатами |
| Q | Выход из программы |

## Алгоритм A*

A* (A-Star) – это алгоритм поиска кратчайшего пути на графе, который использует эвристику для направления поиска. Основные компоненты:

- **G-cost** – стоимость пути от стартовой точки до текущего узла (реальное расстояние)
- **H-cost** – эвристическая оценка расстояния от текущего узла до цели (манхэттенское расстояние)
- **F-cost** = G-cost + H-cost – полная оценочная стоимость пути через текущий узел

Алгоритм всегда выбирает узел с наименьшим F-cost для дальнейшего исследования. При равенстве F-cost выбирается узел с меньшим H-cost (ближе к цели).

Условные обозначения
Символ	Название	Значение
S	Start	Стартовая точка
G	Goal	Целевая точка
@	Current	Текущий узел (обрабатывается в данный момент)
?	Open	Открытый узел (будет рассмотрен в следующих шагах)
.	Closed	Закрытый узел (уже обработан)
*	Path	Клетка, входящая в найденный кратчайший путь
"#" Wall	Стена (непроходимая клетка)

Цветовая схема
Цвет	Фон	Элемент	Значение
Зеленый	bg_green	S	Стартовая точка
Красный	bg_red	G	Целевая точка
Желтый	bg_yellow	@	Текущий узел
Бирюзовый	bg_cyan	?	Открытые узлы (будут рассмотрены)
Синий	bg_blue	.	Закрытые узлы (уже обработаны)
Зеленый	bg_green	*	Найденный путь
Серый	bright_black	#	Стены
Серый (приглушенный)	dim	.	Пустые непосещенные клетки

Режимы отображения
Режим	Клавиша	Отображаемая информация
Simple	S	Символы: S, G, @, ?, ., *, #
G-cost	G	Числовые значения G (расстояние от старта)
H-cost	H	Числовые значения H (эвристика до цели)
F-cost	F	Числовые значения F = G + H (полная стоимость)
